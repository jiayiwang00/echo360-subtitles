# Echo360 Subtitle Translator

这是一个简单的 Chrome 扩展脚本，用于在 Echo360 页面上读取 `.vtt` 字幕文件，并在视频播放时显示双语字幕。

Note: This code was generated by AI [ChatGPT/Codex]

## 手动安装 Chrome 插件的步骤

1. 准备文件：下载并解压插件文件到您电脑上。
2. 打开“扩展程序”页面：打开 Chrome 浏览器，并在地址栏中输入 chrome://extensions ，或者点击三个点菜单 > 扩展程序 > 管理扩展程序。
3. 启用开发者模式：在页面顶部右角切换开发者模式开关。
4. 加载插件：点击出现的“加载未打包”按钮。选择文件夹：选择包含插件文件的文件夹（必须包含 manifest.json 文件）。

## 功能简介

- 自动查找当前页面的视频和字幕资源
- 将英文字幕翻译成中文并显示为双语字幕
- 使用缓存减少重复翻译请求
- 按当前播放位置优先翻译附近字幕
- 可展开的进度面板，查看翻译状态

## 适用范围

当前扩展只在 `https://echo360.net.au/*` 页面下生效。

## 使用方式

1. 将项目作为 Chrome 扩展加载
2. 打开 Echo360 视频页面
3. 等待脚本自动加载字幕并开始翻译
4. 页面右上角可打开 Translator 面板查看进度

## 停止脚本

如果需要手动停止，可以在页面控制台执行：

```js
window.__subtitleTranslatorCleanup()
```
